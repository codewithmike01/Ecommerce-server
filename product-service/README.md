### Product Service

- [Back to Home](/README.md)

###### Requiremeents

- Docker
- Node
- Npm or Yarn (preferrably)

##### How too start up

- Navigate to the product-service folder

```cms
cd product-service
```

- Install packages

```cmd
yarn
```

- Run server

```cmd
yarn start
```

##### How to Access Endpoint Documentation

- Go to your broswer and paste the link below

```cmd
   http://localhost:4001/api-docs
```
